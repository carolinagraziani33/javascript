# 🌍 Sobre o projeto

Este projeto é um **conversor de moedas e criptomoedas** que permite visualizar valores atualizados, histórico de conversões e informações adicionais sobre cada moeda.

O objetivo foi praticar:

* Manipulação de **DOM**
* Consumo de **APIs externas**
* Criação de **gráficos dinâmicos**
* Armazenamento de dados no navegador
* Desenvolvimento de **interfaces interativas**

---

# ⚙️ Funcionalidades

✔ Conversão entre moedas internacionais
✔ Conversão de criptomoedas (Bitcoin, Ethereum e Dogecoin)
✔ Gráfico histórico de cotação
✔ Tendência de criptomoedas (alta / baixa / estável)
✔ Histórico das últimas conversões
✔ Curiosidades sobre países e moedas
✔ Cidades ecológicas relacionadas às moedas
✔ Modo claro e modo escuro
✔ Dados salvos com **localStorage**
